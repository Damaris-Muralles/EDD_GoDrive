EDD GoDrive es una aplicación Web que busca tener un funcionamiento similar a Google Drive, permitiendo el manejo de archivos para la Facultad de Ingenieria proporcionando a cada uno de los cursos de la carrera un espacio de almacenamiento en donde se puede cargar, crear y eliminar carpetas y archivos.
